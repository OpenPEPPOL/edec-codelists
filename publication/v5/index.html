OpenPEPPOL eDEC Code Lists

The code lists presented here, are referenced by the PEPPOL Policy for use of Identifiers and are mandatory in the PEPPOL network.
The latest code list release is v5 from 2019-08-15. See the Change Log for details.

Usage notes

It is important to note that these are dynamic lists. Over time new services will be added. Developers should take this into account when designing and implementing solutions for PEPPOL services.

Participant identifier schemes

Rows marked as "deprecated" should not be used for newly issued documents, as the respective identifier issuing agency is no longer active/valid. Deprecated scheme IDs may however not be reused for different agencies as existing exchanged documents may refer to them.

Document types

Rows marked as "deprecated" should not be used for newly issued documents.

Rows marked as "deprecated" should not be used for newly issued documents.
